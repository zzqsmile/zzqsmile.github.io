MySQL_布尔注入

1.布尔注入常用函数介绍

  • substr()函数

作用：截取字符串
用法：
substr(string string,num start,num length);
string为字符串
start为起始位置
length为长度

  • ascii()函数

作用：返回字符串str的字符ASCII码值。如果str值是空字符串，返回0，如果str值是NULL，返回NULL。

  • char()

作用：ASCII码值转字符

2.注入过程

2.1猜库

  • 判断数据库个数
  • 逐个判断数据库长度

例如判断第1个数据库information_schema长度为18:

  • 判断第一个库第1个字符的ASCII码值
  • 判断第一个库第2个字符的ASCII码值
  • 判断当前数据库长度
  • 判断当前数据库第1个字符的ASCII码值

2.2猜表

  • 判断当前数据库中表的个数
  • 判断当前数据库中第1个表的长度
  • 判断当前数据库中第1个表的第一个字符的ASCII码值

2.3猜字段

  • 判断当前数据库中第1个表的字段个数
  • 判断当前数据库中第1个表的第1个字段长度
  • 判断当前数据库中第1个表的第1个字段第1个字符的ASCII码值

2.4猜内容

  • 判断表的数据量
  • 判断表中第一行数据长度
  • 判断表中第一行数据第一个字符的ASCII码值
